# Egyptian Mouse Pounce

[Egyptian Rat Screw](https://en.wikipedia.org/wiki/Egyptian_Ratscrew), with more cats.

Built with [Flutter](https://flutter.dev), and runs on Android, iOS, web, macOS, Linux, and probably Windows.
